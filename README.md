## Local Debug Steps

1. `pip install flask`
2. `pip install flask_cors`
3. `pip install bson`
4. `pip install pymongo`
5. `set FLASK_APP=advocateSimpleServer.py`
6. `flask run`
7. Open browser and test REST APIs (may use Chrome extension *Restlet Client*).
